Normalizing using ImageNet-style channel-wise normalization
Normalization Steps
Load image using OpenCV
→ Image is in BGR format (Blue-Green-Red).

Convert BGR to RGB
→ Use cv2.cvtColor(img, cv2.COLOR_BGR2RGB)
→ Now the image is in the standard RGB format.

Resize the image
→ Resize to a fixed size like (224, 224) using cv2.resize.

Scale pixel values to [0, 1]
→ Convert image to float and divide by 255:
img = img.astype(np.float32) / 255.0

Apply ImageNet-style normalization
→ For each channel (R, G, B):
img = (img - mean) / std
where:

mean = [0.485, 0.456, 0.406]

std = [0.229, 0.224, 0.225]
